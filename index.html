Session 1 / AI Video Editing

AI 视频剪辑从生成到交付

掌握生成式 AI 视频全流程，从策划到剪辑的完整工作链路。

核心问题

有了产品 brief，生成一条可预览的 AI 视频需要几步？

传统方式 策划 → 拍摄 → 剪辑 → 后期，周期长、成本高
AI 方式 策划 → 提示词 → 文生图 → 文生视频 → 剪辑 → 交付

标准六步流程

① 视频策划 → ② 撰写提示词 → ③ 文生图 → ④ 文生视频 → ⑤ 剪辑 → ⑥ 交付

各步骤关键产出

视频策划分镜脚本、视觉风格定义
撰写提示词文生图 & 文生视频的 Prompt
文生图分镜关键帧图片（即梦/可灵/星流）
文生视频单镜头视频片段（即梦/可灵/vidu）
剪辑完整成片（剪映/AI剪辑）
交付按客户要求输出最终视频文件

常用平台对比

即梦❌ 不支持真人直接上传｜自研模型｜动画/虚拟角色/风格化视频
可灵✅ 支持真人上传｜自研模型｜真实人物驱动/换脸/数字人
星流Midjourney v7、Image 1.5｜高质量风格化图片生成

平台趋势

目前平台开始接入插件、鼓励制作 Skill、引入 3D 模型进行人物动作和空间关系的固定。

消耗的积分必然更高，但在会使用的情况下，稳定性和质量都会更好。

场景化平台推荐

🎭 皮克斯/3D 动画广告即梦（图片+视频）｜3D 动画质感最好，中文 prompt 友好，高速运动表现优良
🧑 真人剧情片/广告可灵（图片+视频）｜唯一支持真人上传，人物较为真实
🎨 高质量风格化海报/关键帧星流（Midjourney v7）｜艺术风格最丰富，关键帧质量上限高
📱 短视频带货/快速出片即梦（全流程）｜出图出视频都快，积分消耗可控
🎬 品牌宣传片（多风格混合）星流出图 + 可灵出视频｜图用 MJ 保证风格质量，视频用可灵保证流畅

提示词万能公式

Seedance 2.0 的提示词遵循以下结构。清晰的结构化提示词能让 AI 精准理解创作意图，避免"写了很长但出图不准"的问题。

主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束

基础版 vs 进阶版

基础版 一位年轻女生在海边慢走，微风拂动头发，微笑看向镜头，黄昏暖光，4K 高清，电影感，稳定运镜，画面流畅不抖动，细节清晰
进阶版公式 [核心主体] 在 [特定场景] 完成 [叙事动作]，[时长+精准运镜+焦点锁定]，[焦段+光圈+画幅]，[布光+核心色调]，[电影胶片型号+调色标准]，[锁种跟焦+动态运镜平滑+元素防穿模+4K]

公式拆解说明

核心主体明确画面主角｜例：一位年轻女生
特定场景提供环境上下文｜例：海边沙滩
叙事动作定义角色行为｜例：慢走、微笑看向镜头
运镜 & 焦点控制镜头运动方式｜例：稳定运镜、焦点锁定面部
焦段 & 光圈决定景深和视角｜例：50mm、f/1.8、16:9
布光 & 色调营造氛围｜例：黄昏暖光、金色侧逆光

不等于 prompt 越长越好。保证【主体】、【场景】、【音乐】（只生成音效）、【镜头】即可。上传过参考图就不需要再花大段文字阐述。

负面提示词（Negative Prompt）

告诉 AI "不要出现什么"，和正面提示词同样重要。AI 模型有默认倾向（完美皮肤、过度饱和、多手指），仅靠正面描述无法抑制。

画质问题畸形手指、多余肢体、面部扭曲、低清模糊｜所有人物生成
风格偏差真人照片感（做动画时）、恐怖氛围、赛博朋克｜风格化项目
文字干扰杂乱乱码文字、品牌 logo、水印｜广告/商业项目
构图问题多余人物、杂乱背景、过度反光塑料感｜产品广告
情绪溢出哭泣崩溃（写疲惫时）、过度煽情、暴力｜情绪叙事
实践建议5-10 个关键词即可，针对每次翻车的具体问题动态添加，不同镜头负面提示词可能不同

模板一：产品广告

适用场景：30 秒产品宣传视频，需要突出产品卖点和使用场景。

[产品名称/类型] 放置在 [具体场景] 中，[核心卖点动作或效果]。 [人物（可选）] 与产品互动：[具体交互动作]。 [光线方案]：[主光源方向 + 色温 + 氛围词]。 [镜头语言]：[焦段] [景别] [运镜方式]，焦点锁定 [产品/人物面部]。 [风格]：[3D动画/写实/电影感] 广告片质感，[色调描述]。 [画质约束]：4K 高清，材质细腻，[产品材质描述]。

产品广告示例（音响）

哑光深灰圆角智能音响放置在现代小书房桌面上，顶部光环亮起发出柔和暖色声波。年轻女性产品经理坐在电脑后方，被声波吸引抬眼。电脑冷光为主光源，音响暖光为辅。35mm 中近景，缓慢推向音响，焦点锁定光环。皮克斯式 3D 动画广告质感，冷蓝环境中出现一点温暖声波。4K 高清，哑光音响材质和蓬松头发清晰可见。

负面 Prompt：真人照片感、恐怖氛围、过暗画面、赛博朋克、杂乱乱码文字、品牌 logo、低质塑料音响、畸形手指、多余人物、过度写实皮肤、强烈噪点、字幕

模板二：品牌宣传片

适用场景：品牌形象片、企业宣传、活动预热，偏氛围和调性。

[品牌调性关键词] 的画面氛围，[核心视觉符号/场景] 作为主体。 [环境描写]：[建筑/自然/室内空间] + [时间/天气/光线]。 [人物群像（可选）]：[角色状态] + [与环境的互动方式]。 [镜头语言]：[焦段] [景别]，[运镜方式]，[节奏：缓慢/快切/呼吸感]。 [色彩体系]：[主色调] + [辅助色] + [情绪色调变化]。 [风格]：[电影感/纪录片/广告片] 质感，[参考导演/影片风格]。 [画质约束]：4K 高清，[胶片型号/调色标准]，画面干净无噪点。

品牌宣传片示例

温暖而现代的品牌氛围画面，城市天际线在黄昏中缓缓亮起灯光。玻璃幕墙建筑反射金色夕阳，街道上行人步履从容。35mm 中景，缓慢横移推近，呼吸感节奏。主色调为暖金色和深蓝过渡，辅助色为城市霓虹的柔和点缀。电影感广告片质感，参考苹果品牌片风格。4K 高清，柯达 2383 胶片调色，画面干净通透。

负面 Prompt：杂乱广告牌、品牌 logo（非自家）、过度饱和、手机拍摄感、抖动画面、文字水印

模板三：MG 动画

适用场景：程序化动画、数据可视化、Logo 演绎、产品功能演示。

[动画类型] 风格的 MG 动画画面，[核心元素/Logo] 作为视觉中心。 [动画动作]：[元素入场方式] → [变形/数据增长/路径运动] → [收束方式]。 [配色方案]：[主色] + [辅助色] + [背景色]，[渐变方向]。 [图形语言]：[几何形状/线条/粒子] + [运动节奏：弹跳/流畅/阶梯式]。 [镜头/画面]：16:9，[居中/黄金分割] 构图，[整体视角/局部特写]。 [风格]：扁平化/2.5D/3D MG 动画，[参考风格]。 [技术约束]：[帧率] [分辨率]，[渲染要求]。

提示词邪修

由于提示词污染或者是 AI 幻觉，需要根据实际情况，讲一些不符合创作的"邪修"提示词。

提示词污染模型训练数据中某些词汇有隐含含义，导致生成偏差
AI 幻觉模型"发挥"出不在提示词中的元素，需要用非常规手段约束

真人视频：调动脸部与肢体动作

初版（太笼统） 女生不屑地看着前方
优化版（肌肉级别） 女生眼睑微压低瞳孔短暂停留。她转头将视线冷淡地扫开，翻了个白眼，下颌轻微又轻抬，嘴角出现极短暂地下压，轻微耸肩，不屑。
  • 将抽象情绪拆解为具体肌肉动作（眼睑、瞳孔、嘴角、下颌、肩膀）
  • 加入时序关系（"短暂停留"→"转头"→"翻白眼"→"下颌轻抬"）
  • 每个动作都给了幅度修饰词（"微压""极短暂""轻微"），避免动作过大

真人视频：让肤质更加真实

初版（塑料感） 图书馆里一位年轻女学生的特写肖像，坐在带台灯的木桌旁，拿着笔的手托着脸看镜头。
优化版（加瑕疵） ...可见的皮肤毛孔，因思考眯眼而产生的眼角自然细纹，熬夜带来的轻微黑眼圈与皮肤瑕疵。
  • 主动添加"瑕疵"描述（毛孔、细纹、黑眼圈），打破 AI 的"完美皮肤"默认倾向
  • 瑕疵与场景逻辑自洽（"熬夜"呼应"图书馆学习"）
  • 用"自然"修饰，避免瑕疵过度导致恐怖谷效应

真人视频：情绪的表现

写悲伤，不一定是直接呈现哭泣。通过"状态描写"传递情绪比直接写"哭"更有感染力。

女生脸上挂着未干的泪痕，眼神是剧烈哭泣后的空洞与麻木，随后，她缓慢且沉重地闭了一下眼睛，像是在隐忍泪意，嘴唇微微颤抖。
痕迹描写"未干的泪痕" → 暗示已经哭过，而非正在哭
眼神状态"空洞与麻木" → 哭泣后的疲惫感
动作节奏"缓慢且沉重地闭了一下眼睛" → 用慢动作传递情绪重量
微表情"嘴唇微微颤抖" → 隐忍的信号，克制比释放更有张力

AI Agent 在工作流中的应用

有了 Agent，哪些步骤可以用 AI 做？

视频策划将公司审美偏好制作成 Skill → 丢一个 brief 给 Agent，自动生成视频策划案
撰写提示词将写提示词的偏好制作成 Skill → Agent 撰写初版提示词
剪辑AI 剪辑能做到什么程度？（当前探索中）Remotion / 剪映 API / Hermes

为什么需要 Skill

一条 30 秒的片子目前大概需要 8-10 个镜头，其中环境、人物形象等元素是高度重复的。

一致性问题同一角色在 8-10 个镜头中需要保持外貌一致
效率问题逐条手写提示词耗时且容易遗漏关键约束
风格统一不同镜头的运镜风格、色调、画质标准需要保持统一

通过制作 Skill，让 Agent 接管这些重复性工作，人只需要审核和微调。

Skill 制作：需求讨论

核心需求：我要做 AI 短视频。视频的分镜图片和生成视频的提示词通常需要 非常详细。我希望有一个 Skill 能够分析我的故事剧情、人物，并为我生成对应 的详细提示词。我的作品风格可能会比较多样，所以我在想是固定一个风格的 Skill，还是在同一个 Skill 中通过关键词切换风格。
厘清交付物：我希望明确——目前的这个流程，它交付的内容是什么？流程的 Skill 走完以后它交付的是什么东西？资产积累不应该放在交付的部分，分析步 骤里夹杂了交付，而分析部分又放在了不正确的位置。请你重新调整一下。

Skill 调优迭代

我尝试跑了一条视频以后，觉得它给的分镜逻辑不是特别好，经常会有空间的 错乱或者是一些动作不衔接的地方。这个是否能给一个提示词去优化它的分镜 分析，还是分镜产出的那一部分 Skill 的包装可以保留更多一点？
故事分析 人物锁定 分镜连续性 提示词产出

Skill 输入：用户提供的原始素材

用户只需要提供一个简单的故事描述。没有分镜编号、没有镜头语言、没有风格参考——这些全部由 Skill 自动补全。

世界观：现代 视频长度：30s 目的：我要做一个皮克斯风格的广告 销售产品：音响 产品故事： 音响能给人身临其境的感觉，开着音响的时候觉得自己在短剧片场当总裁。 被人关了音响才发现自己在做产品的 ppt 的时候睡着了。 回到蓝色调的忧郁色调的房间。 妈妈为她切换了一首温馨的音乐，房间变成暖黄色的，她伸了个懒腰继续做 ppt。 人物： 女主：长头发卷发，产品经理。短剧场景里，她是大女主形象。

Skill 输出：AI短片 Prompt 包

① 项目总览项目名称、一句话概括、核心情绪、风格方向、镜头数量、预计时长
② 输入内容整理人物/场景/道具/风格参考的结构化整理，标注缺失信息和系统补全
③ 故事与风格分析每个镜头的叙事功能、角色目标、情绪目标、视觉重点 + 风格翻译结果
④ 连续性报告空间/人物位置/人物状态/道具状态/时间的连续性台账 + 问题及修正建议
⑤ 角色锁/世界锁/风格锁外貌/服装/气质/动作习惯锁定；建筑/道具/色彩锁定；摄影/灯光/质感锁定
⑥ 标准镜头卡每个镜头的景别/构图/光线/焦段/运镜/时长/声音提示/连续性状态
⑦ 图片 Prompt 包中文+英文图片提示词 + 负面提示词 + 各平台建议 + Source Trace
⑧ 视频 Prompt 包中文+英文视频提示词 + 运动控制 + 负面提示词 + 平台适配 + Source Trace
⑨ QA 报告每个镜头的通过项/风险项/修改建议/连续性检查/注入检查

Source Trace 来源追溯

每条提示词都带来源追溯，确认 Character/World/Style/Shot 四个维度是否都正确注入，防止"只写了画面但漏了角色设定"。

Character 来源年轻职场女性产品经理、长卷发、大眼睛、圆润脸颊、浅色居家针织衫、眼皮微沉
World 来源现代城市公寓小书房、冷蓝夜晚、笔记本电脑、哑光深灰圆角智能音响
Style 来源皮克斯式 3D 动画广告、35mm 中近景、冷蓝环境与暖色光环、干净细腻材质
Shot 来源音响启动、第一圈暖色光环、小妞疲惫做 PPT、连续性状态

Skill 制作经验总结

需求定义明确"交付物是什么"比"功能列表"更重要
流程设计确保输入 → 分析 → 交付的顺序正确，不要把交付混入分析步骤
